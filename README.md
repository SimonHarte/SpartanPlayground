SpartanPlayground
================================

## Requirements

### Install Node

[nodejs.org](http://nodejs.org/)

### Install XTC CLI

```
npm i -g xtc-cli
```

## Startup

### Run XTC Server

```
xtc start
```

### Run FE Build

```
xtc build
```

### Open in Browser

[`localhost:3000`](http://localhost:3000)
